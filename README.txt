Hexen II: Hammer of Thyrion (uHexen2) source code.
http://uhexen2.sourceforge.net/
http://sourceforge.net/projects/uhexen2/

Guide:
1. Install the Win32 version of Hammer of Thyrion 1.5.9 from SourceForge
2. Get the Release from this repository and unzip with overwrite over the HoT installation
  Note: glh2.exe, a bunch of dll and the autoexec.cfg will be overwritten.
3. Get Remix Runtine for x86 games, and unzip in the HoT folder
4. Run the game with glh2.exe
